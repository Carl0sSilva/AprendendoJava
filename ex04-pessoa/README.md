# Encapsulamento e Construtores

Este exercício foca na segurança e integridade dos dados dentro da Programação Orientada a Objetos.

## 🧠 Conceitos Chave

- **Encapsulamento:** Uso do modificador `private` para impedir o acesso direto aos atributos.
- **Getters & Setters:** Métodos públicos que permitem ler (`get`) e modificar (`set`) os valores privados com segurança.
- **Construtor:** Implementação de um método especial para garantir que o objeto seja instanciado com estados iniciais obrigatórios (neste caso, o nome).

## 🛠️ Como executar

1.  **Acesse o diretório:**
    ```bash
    cd ex04-pessoa
    ```

2.  **Compile:**
    ```bash
    javac *.java
    ```

3.  **Inicie:**
    ```bash
    java MyClass
    ```
