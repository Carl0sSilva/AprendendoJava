# Instanciação de Objetos: Classe Carro

Este exercício demonstra a base da Programação Orientada a Objetos: o uso de uma **Classe** como molde para a criação de múltiplos **Objetos**.

## 🧠 Conceitos Chave

- **Classes vs Objetos:** A classe `Carro` define a estrutura, enquanto `carro1`, `carro2` e `meuCarro` são instâncias reais com dados próprios.
- **Independência de Estado:** Alterar o ano do `carro1` não afeta o `carro2`, provando que cada objeto ocupa seu próprio espaço na memória.
- **Métodos de Exibição:** Uso de `System.out.printf` para formatação de dados numéricos e strings.

## 🛠️ Como executar

1.  **Navegue até a pasta:**
    ```bash
    cd ex03-carros
    ```

2.  **Compile:**
    ```bash
    javac *.java
    ```

3.  **Execute:**
    ```bash
    java MyClass
    ```
