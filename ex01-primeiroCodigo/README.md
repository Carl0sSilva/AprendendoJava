# Contador Interativo em Java

Este projeto inicial foca na sintaxe básica do Java, manipulação de tipos primitivos e entrada de dados via argumentos do sistema.

## 🧠 Conceitos Aplicados

* **Loop `while`**: Estrutura de repetição para contagem.
* **Argumentos (`args`)**: Captura de dados passados ao executar o programa no terminal.
* **Conversão de Tipos**: Uso de `Integer.parseInt` para tratar a entrada.

## 🛠️ Como executar

1.  **Entre na pasta:**
    ```bash
    cd ex01-primeiroCodigo
    ```

2.  **Compile:**
    ```bash
    javac Main.java
    ```

3.  **Execute (substitua o '5' pelo número desejado):**
    ```bash
    java Main 5
    ```
