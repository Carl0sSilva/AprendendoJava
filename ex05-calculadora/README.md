# Calculadora Básica

Este exercício demonstra a implementação de operações matemáticas simples através de métodos com passagem de parâmetros.

## 🧠 Conceitos Chave

- **Parâmetros de Método:** Demonstração de como enviar dados para dentro de uma função.
- **Operações Aritméticas:** Prática com soma, subtração, multiplicação e divisão.
- **Formatação de Strings:** Uso do `printf` para criar mensagens dinâmicas e legíveis no console.

## 🛠️ Como executar

1.  **Navegue até a pasta:**
    ```bash
    cd ex05-calculadora
    ```

2.  **Compile:**
    ```bash
    javac *.java
    ```

3.  **Execute:**
    ```bash
    java MyClass
    ```
